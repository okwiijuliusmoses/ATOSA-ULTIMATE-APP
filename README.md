# ATOSA - Alumni Mobile App

Welcome to the ATOSA Alumni Mobile Application repository!

## Quick Start

Open `index.html` in your web browser to see the interactive ATOSA app prototype.

## Features

✅ **Home Dashboard** - View member info and quick access menu
✅ **Membership Card** - Display digital membership card with QR code
✅ **Payments** - Make payments via mobile money
✅ **SACCO** - Check SACCO account balance
✅ **Events** - View upcoming alumni events
✅ **Documents** - Access certificates and transcripts
✅ **Settings** - Manage app preferences

## How to Use

1. Download the repository
2. Open `index.html` in your browser
3. Click on different menu items to explore all features
4. Use the bottom navigation to switch between screens

## To Download APK (Native Android App)

For the native Android app, follow these steps:

1. Clone this repository
2. Install Flutter SDK
3. Run: `flutter pub get`
4. Run: `flutter build apk --debug`
5. The APK will be in: `build/app/outputs/flutter-apk/app-debug.apk`

## Technology Stack

- Frontend: HTML5, CSS3, JavaScript
- Mobile: Flutter (Dart)
- Backend: NestJS (TypeScript)
- Database: PostgreSQL
- API: Custom REST API
